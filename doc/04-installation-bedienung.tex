\section{Installation und Bedienung}
\subsection{Installation}
Der Beleg wird als Zip-Archiv abgegeben, welches lediglich an eine beliebige Stelle im 
Dateisystem entpackt werden muss. Darin befindet sich im Ordner ''bin'' bereits eine 
unter Windows lauffähige Fassung des Belegs sowie die Visual-Studio-Projektmappe 
(cg1-beleg.sln) zur eigenständigen Kompilierung.

Die notwendigen Drittbibliotheken / Abhängigkeiten sind im Ordner ''lib'' mitgeliefert.
Da die Anwendung nur im ''Release''-Modus zufriedenstellend performant ist 
(siehe ''Probleme'') wurden die Abhängigkeiten noch einmal im Release-Modus aus ihren 
Quellen kompiliert und im Ordner ''contrib'' abgelegt.

\subsection{Bedienung}
Das Programm ist selbsterklärend. Die Bedienung wird durch ein Overlay auf dem 
Bildschirm erklärt. Desweiteren ist die grundsätzliche Bedienung noch einmal im 
Abschnitt ''Lösungsansatz'' geschildert.
